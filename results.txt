Motor-Balance PD-Referenced Classification — Results Summary

PRIMARY DESIGN (MAIN CLAIM)
Train PD-only H&Y reference models on shared balance scales (BBS, Mini-BEST, FES) using
leave-one-out cross-validation (LOOCV). Save the best-performing pipelines (macro-F1 on
the Combined feature set). Apply the multiclass pipeline to welders to obtain PD-like stage
probabilities and a continuous PD-like severity score. This is a resemblance framework, not
a PD diagnosis and not a true clinical H&Y assignment in welders.

SUPPORTING BENCHMARK (NOT THE H&Y MODEL)
PD vs Welder group discrimination on the same three balance features: stratified 5-fold CV,
n=30 (14 vs 16). Metrics in outputs/metrics/group_discrimination.json; figure
fig_08_group_discrimination.png. Confounded by age and cohort — descriptive / supporting only.

SECONDARY ANALYSIS (EXPLORATORY)
Associations between PD-like severity score and welding exposure variables (e.g. years,
PPE) are optional, hypothesis-generating, and must not overshadow the primary PD-reference
design. See outputs/metrics/phase2_associations.json.

INPUT FEATURES
  1. BBS (Berg Balance Scale)
  2. Mini-BEST (Mini-BESTest)
  3. FES (Falls Efficacy Scale)

MODELS COMPARED
Logistic regression, random forest, gradient boosting; preprocessing: median imputation,
standardization, classifier — inside each LOOCV fold.

Representative replication snapshot (Random Forest, Combined, current dataset):
  • Binary LOOCV accuracy ≈ 0.857; macro-F1 ≈ 0.844
  • Multiclass LOOCV exact accuracy ≈ 0.50; within-one-stage ≈ 1.00

WELDER OUTPUTS (PHASE 2)
Per row: Pred_Stage, P_Stage1…P_Stage4 (as trained), PD_Severity_Score = Σ k·P(stage k);
optional binary Early/Late-like labels if binary pipeline is applied.

  • Pipeline figures: outputs/figures/fig_01 … fig_07
  • Paper-ordered figures + captions: outputs/figures/paper/paper_fig_*.png and .md

STRENGTHS
Shared PD–welder features; LOOCV for small PD samples; probabilistic outputs; scripted
reproducibility; generic inference via python -m src.predict_excel

EXTERNAL VALIDATION (REQUIRED FOR STRONGER CLAIMS; NOT IN THIS PILOT)
The PD reference model should be evaluated on an independent PD cohort with true H&Y and
the same balance measures, using frozen pipelines (no retraining on the external set for
the primary estimate). Welders cannot validate H&Y accuracy (no true H&Y labels).
When external data are unavailable, state explicitly that findings are hypothesis-generating.
See docs/external_validation.md. When data exist: python -m src.validate_external_pd <file.xlsx>

LIMITATIONS
n=14 PD; H&Y imbalance; no external PD cohort in this study; age gap between cohorts;
three scales only; welder outputs are non-diagnostic and exploratory.

RECOMMENDED REPORTING
This study presents a PD-referenced machine learning framework in which models trained on
PD H&Y severity using BBS, Mini-BEST, and FES are applied to welders to characterize
PD-like motor-balance severity resemblance categories using multiclass probabilities and a
severity score.
